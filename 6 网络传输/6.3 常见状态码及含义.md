| 状态码分类         | 状态码 | 英文标识                  | 核心含义                  | 实战说明                                |
| ------------- | --- | --------------------- | --------------------- | ----------------------------------- |
| 信息性状态码（1xx）   | 100 | Continue              | 服务器已接收请求头，客户端可继续发送请求体 | 大文件上传场景，避免一次性发大量数据超时                |
| 信息性状态码（1xx）   | 101 | Switching Protocols   | 服务器同意协议升级             | 自动触发，如 HTTP/1.1 升 HTTP/2，前端无需处理     |
| 成功状态码（2xx）    | 200 | OK                    | 通用成功，服务器正常返回结果        | 最常用，适用于查询、登录成功等所有正常请求               |
| 成功状态码（2xx）    | 204 | No Content            | 请求成功，无响应体             | 删除资源、OPTIONS预检等场景，节省带宽              |
| 成功状态码（2xx）    | 206 | Partial Content       | 服务器部分返回资源             | 大文件下载、视频断点续传，需带Range请求头             |
| 重定向状态码（3xx）   | 301 | Moved Permanently     | 永久重定向，资源迁移新URL        | 域名/旧接口迁移，浏览器永久缓存新地址                 |
| 重定向状态码（3xx）   | 302 | Found                 | 临时重定向，资源临时迁移新URL      | 活动页跳转、未登录跳登录页，不缓存新地址                |
| 重定向状态码（3xx）   | 304 | Not Modified          | 协商缓存命中，资源未修改          | 配合ETag/Last-Modified，前端用本地缓存优化性能    |
| 客户端错误状态码（4xx） | 400 | Bad Request           | 请求参数错误，服务器无法解析        | 参数缺失/格式错、JSON语法错、URL特殊字符未编码         |
| 客户端错误状态码（4xx） | 401 | Unauthorized          | 未授权/身份过期，需重新登录        | 无Token/Token失效/Session过期，区别403（未认证） |
| 客户端错误状态码（4xx） | 403 | Forbidden             | 禁止访问，已认证但无权限          | 权限不足、IP白名单限制、接口被封禁                  |
| 客户端错误状态码（4xx） | 404 | Not Found             | 资源不存在，请求URL无效         | URL拼写错、接口下线、资源ID不存在                 |
| 客户端错误状态码（4xx） | 405 | Method Not Allowed    | 请求方法不被支持              | 方法用错、后端未配置；响应头Allow显示支持方法           |
| 客户端错误状态码（4xx） | 408 | Request Timeout       | 请求超时，服务器等待过久          | 网络卡顿、传输慢；可增加前端超时配置                  |
| 客户端错误状态码（4xx） | 429 | Too Many Requests     | 请求频繁，触发限流             | 未做防抖、超阈值；响应头Retry-After提示重试时间       |
| 服务器错误状态码（5xx） | 500 | Internal Server Error | 服务器内部通用错误             | 代码bug、未捕获异常、业务逻辑出错                  |
