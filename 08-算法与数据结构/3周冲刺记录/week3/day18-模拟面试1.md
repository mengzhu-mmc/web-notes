# Day 18 (03-24) — DP 复习

## 算法题

### 1. [322] 零钱兑换 — Medium
- 链接：https://leetcode.cn/problems/coin-change/
- 思路：完全背包，`dp[i] = min(dp[i], dp[i - coin] + 1)`；dp 初始化为 `Infinity`，`dp[0] = 0`

### 2. [300] 最长递增子序列 — Medium
- 链接：https://leetcode.cn/problems/longest-increasing-subsequence/
- 思路：O(n²) DP：`dp[i]` 表示以 `nums[i]` 结尾的 LIS 长度；贪心 + 二分可优化到 O(n log n)

### 3. [1143] 最长公共子序列 — Medium
- 链接：https://leetcode.cn/problems/longest-common-subsequence/
- 思路：二维 DP，`dp[i][j]` 表示 `text1[0..i-1]` 与 `text2[0..j-1]` 的 LCS 长度；`text1[i-1]===text2[j-1]` 时 `dp[i][j] = dp[i-1][j-1] + 1`，否则取两者最大

### 4. [518] 零钱兑换 II — Medium
- 链接：https://leetcode.cn/problems/coin-change-ii/
- 思路：完全背包组合数，外层遍历硬币，内层遍历金额（顺序遍历）；`dp[i] += dp[i - coin]`

## 知识点：HTTPS/TCP + 跨域（复习）
- **HTTPS**：TLS 握手（证书校验 → 协商密钥 → 对称加密通信）；证书防中间人攻击
- **TCP 三次握手**：SYN → SYN+ACK → ACK；四次挥手：FIN → ACK → FIN → ACK
- **跨域**：浏览器同源策略限制；解决方案：CORS（`Access-Control-Allow-Origin`）/ 代理服务器 / JSONP
- **CORS 简单请求 vs 预检**：非简单请求先发 OPTIONS 预检

## 核心套路
> **DP 四步法**：① 定义状态 ② 写转移方程 ③ 初始化边界 ④ 确定遍历顺序。  
> **背包口诀**：0/1 背包逆序遍历容量；完全背包顺序遍历容量；求排列外循环容量，求组合外循环物品。
