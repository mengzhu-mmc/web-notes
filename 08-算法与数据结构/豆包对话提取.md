## 编辑距离（LeetCode 72）

### 核心知识点
- 典型动态规划问题（Levenshtein距离）
- `dp[i][j]` 定义：word1 前 i 个字符转为 word2 前 j 个字符的最少操作数
- **易错点**：访问字符串时用 `word1[i-1]` 而非 `word1[i]`

### 状态转移
- 相等：`dp[i][j] = dp[i-1][j-1]`
- 不等：`dp[i][j] = Math.min(替换dp[i-1][j-1], 删除dp[i-1][j], 插入dp[i][j-1]) + 1`

### 代码（二维DP）
```js
var minDistance = function(word1, word2) {
  const m = word1.length, n = word2.length;
  const dp = new Array(m+1).fill(0).map(_ => new Array(n+1).fill(0));
  for(let i = 0; i <= m; i++) dp[i][0] = i;
  for(let i = 0; i <= n; i++) dp[0][i] = i;
  for(let i = 1; i <= m; i++) {
    for(let j = 1; j <= n; j++) {
      if (word1[i-1] === word2[j-1]) {
        dp[i][j] = dp[i-1][j-1];
      } else {
        dp[i][j] = Math.min(dp[i][j-1], dp[i-1][j], dp[i-1][j-1]) + 1;
      }
    }
  }
  return dp[m][n];
};
```

### 空间优化（一维DP）
- 只保留一行，用 `prev` 变量存"左上角 dp[i-1][j-1]"
- 三变量：`dp[j]`旧=上，`dp[j-1]`新=左，`prev`=左上角
```js
var minDistance = function(word1, word2) {
  const m = word1.length, n = word2.length;
  const dp = new Array(n + 1).fill(0);
  for (let j = 0; j <= n; j++) dp[j] = j;
  for (let i = 1; i <= m; i++) {
    let prev = dp[0];
    dp[0] = i;
    for (let j = 1; j <= n; j++) {
      const temp = dp[j];
      const left = dp[j-1], up = dp[j], lu = prev;
      dp[j] = word1[i-1] === word2[j-1] ? lu : Math.min(left, up, lu) + 1;
      prev = temp;
    }
  }
  return dp[n];
};
```

## 最大子数组和（LeetCode 53）

### Kadane算法（最优 O(n) O(1)）
```js
var maxSubArray = function(nums) {
  let curSum = nums[0], maxSum = nums[0];
  for (let i = 1; i < nums.length; i++) {
    curSum = Math.max(nums[i], curSum + nums[i]);
    maxSum = Math.max(maxSum, curSum);
  }
  return maxSum;
};
```

### 方法对比
| 方法 | 时间 | 空间 |
|------|------|------|
| 暴力 | O(n²) | O(1) |
| 前缀和 | O(n) | O(n) |
| 分治 | O(nlogn) | O(logn) |
| Kadane | O(n) | O(1) ✅ |

## 乘积最大子数组（LeetCode 152）

### 核心：同时维护最大值和最小值（因为负×负=正）
```js
var maxProduct = function(nums) {
  let curMax = nums[0], curMin = nums[0], res = nums[0];
  for (let i = 1; i < nums.length; i++) {
    const tempMax = curMax;
    curMax = Math.max(nums[i], tempMax * nums[i], curMin * nums[i]);
    curMin = Math.min(nums[i], tempMax * nums[i], curMin * nums[i]);
    res = Math.max(res, curMax);
  }
  return res;
};
```

### 面试要点
- `curMax/curMin` = 以当前元素结尾的子数组的最大/最小乘积
- 必须保存 `tempMax`，因为更新 curMin 时需要原始 curMax

## 最长回文子串（LeetCode 5）

### 中心扩展法（推荐，O(n²) O(1)）
```js
var longestPalindrome = function(s) {
  if(s.length <= 1) return s;
  let res = '';
  for(let i = 0; i < s.length; i++) {
    handle(i, i);   // 奇数
    handle(i, i+1); // 偶数
  }
  function handle(i, j) {
    while(i >= 0 && j < s.length && s[i] === s[j]) { i--; j++; }
    const temp = s.slice(i+1, j);
    res = res.length > temp.length ? res : temp;
  }
  return res;
};
```

### DP解法（O(n²) O(n²)）
- `dp[i][j]`：s[i..j] 是否为回文
- 转移：`s[i]===s[j]` 时，`j-i<3` 直接true，否则看 `dp[i+1][j-1]`

## 最长公共子序列（LeetCode 1143）

### 状态转移
- `dp[i][j]` = text1前i个与text2前j个的LCS长度
- 相等：`dp[i][j] = dp[i-1][j-1] + 1`
- 不等：`dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1])`

```js
function longestCommonSubsequence(text1, text2) {
  const m = text1.length, n = text2.length;
  const dp = Array.from({length: m+1}, () => Array(n+1).fill(0));
  for (let i = 1; i <= m; i++) {
    for (let j = 1; j <= n; j++) {
      if (text1[i-1] === text2[j-1]) dp[i][j] = dp[i-1][j-1] + 1;
      else dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1]);
    }
  }
  return dp[m][n];
}
```

## 最长递增子序列（LeetCode 300）

### DP解法 O(n²)
```js
var lengthOfLIS = function(nums) {
  const dp = new Array(nums.length).fill(1);
  let maxLen = 1;
  for (let i = 1; i < nums.length; i++) {
    for (let j = 0; j < i; j++) {
      if (nums[i] > nums[j]) dp[i] = Math.max(dp[i], dp[j] + 1);
    }
    maxLen = Math.max(maxLen, dp[i]);
  }
  return maxLen;
};
// 为什么用maxLen而不是dp[n-1]：LIS不一定以最后元素结尾
```

### 贪心+二分 O(nlogn) ✅ 最优
```js
var lengthOfLIS = function(nums) {
  if (!nums.length) return 0;
  const tails = [nums[0]];
  for (let i = 1; i < nums.length; i++) {
    if (nums[i] > tails[tails.length-1]) {
      tails.push(nums[i]);
    } else {
      let l = 0, r = tails.length - 1;
      while (l < r) {
        const mid = (l + r) >> 1;
        if (tails[mid] < nums[i]) l = mid + 1;
        else r = mid;
      }
      tails[l] = nums[i]; // 注意是 tails[mid] < nums[i]
    }
  }
  return tails.length;
};
```

### 贪心理解
- `tails[k]` = 长度为 k+1 的LIS的**最小末尾值**
- tails 始终严格递增（二分前提）
- 替换操作：不改变当前长度，只优化"末尾更小"，为后续接棒创造机会
- tails 不是真实LIS，只是长度相等
- 示例 `[1,3,4,5,2]`：tails最终 `[1,2,4,5]`，长度4（真实LIS是`[1,3,4,5]`）

## 分割等和子集（LeetCode 416）

### 核心：0-1背包
- **0-1背包遍历顺序**：先遍历物品，再**倒序**遍历容量
- 倒序原因：确保每个元素只被选一次

```js
function canPartition(nums) {
  const sum = nums.reduce((a, b) => a + b, 0);
  if (sum % 2 !== 0) return false;
  const target = sum / 2;
  const dp = new Array(target + 1).fill(false);
  dp[0] = true;
  for (let num of nums) {
    for (let i = target; i >= num; i--) {  // 倒序！
      dp[i] = dp[i] || dp[i - num];
      // dp[i]=不选num, dp[i-num]=选num
    }
  }
  return dp[target];
}
```

## JS 遍历 Set 的方法

```js
const mySet = new Set([1, 2, 3]);

// 1. for...of（最推荐）
for (const item of mySet) { console.log(item); }

// 2. forEach
mySet.forEach((value) => { console.log(value); });

// 3. 转数组
[...mySet].forEach(item => console.log(item));
Array.from(mySet).forEach(item => console.log(item));

// 4. 迭代器方法（values/keys/entries）
for (const item of mySet.values()) { console.log(item); }
```

### 面试要点
- Set 的 `forEach` 回调前两个参数值相同（没有索引）
- `keys()` 和 `values()` 效果完全一致（Set没有键）
