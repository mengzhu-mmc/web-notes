## 一、HTML & CSS 基础 ⏱ 1~2h

> 面试必考但篇幅小，快速过完即可

**文件路径：**
- `01-HTML-CSS/HTML基础/HTML5的新特性.md`
  → 语义化标签、新 API（Canvas/Video/Storage）
- `01-HTML-CSS/CSS选择器与优先级.md`
  → 权重计算、伪类伪元素、层叠规则
- `01-HTML-CSS/CSS基础/BFC详解.md`
  → BFC 触发条件、应用场景（清浮动、隔离外边距）
- `01-HTML-CSS/CSS基础/CSS基础知识点.md`
  → 盒模型（content-box vs border-box）、替换元素、link vs @import、响应式方案
- `01-HTML-CSS/CSS基础/CSS性能优化属性详解.md`
  → will-change、transform、contain

**核心考点：**
- BFC 是什么、怎么触发、解决什么问题
- 盒模型两种区别、border-box 为什么现代推荐
- CSS 选择器优先级计算（0-0-0-0 权重）
- 响应式三种方案对比（媒体查询 / rem / vw）

  → 事件冒泡/捕获、事件委托、stopPropagation vs preventDefault

**核心考点：**
- 手写 `instanceof`、`new`、`call/apply/bind`（见第六章）
- EventLoop 输出顺序题（必考！）
