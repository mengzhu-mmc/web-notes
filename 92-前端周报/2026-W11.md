## 🎨 张鑫旭博客本周更新

### 1. HTML dialog 元素新支持 closedBy 属性（3.9）
**链接：** https://www.zhangxinxu.com/wordpress/2026/03/html-dialog-closedby/

`<dialog>` 新增 `closedBy` 属性，支持点击蒙层关闭模态框，无需 JS：

```html
<dialog closedBy="any">点击蒙层我会隐藏哦~</dialog>
```

**三个枚举值：**
- `any` — 点击外部 / ESC / close() 均可关闭
- `closerequest` — ESC / close() 可关闭，点击外部不行
- `none` — 仅 close() 或表单提交可关闭

**兼容性：** Safari 暂不支持，可用 [dialog-closedby-polyfill](https://github.com/tak-dcxi/dialog-closedby-polyfill)

### 2. HTML interestfor 属性与悬停 popover 交互（3.4）
**链接：** https://www.zhangxinxu.com/wordpress/2026/03/css-interestfor-invoker-target-source/

原生 hover 驱动的 popover 交互，无需 JS：

```html
<button interestfor="imgBook">Hover 显示图片</button>
<img id="imgBook" popover src="book.jpg" />
```

**亮点：**
- 支持 `<a>` 和 `<area>` 元素（popovertarget 仅限 `<button>`）
- 非 popover 元素也能用（配合 `:interest-target` 伪类做高亮等效果）
- 新增 CSS 伪类：`:interest-source` / `:interest-target`
- **兼容性：** 仅 Chrome 142+
