## 🌟 本周焦点

### 1. Chrome 146 推出 DBSC — Cookie 盗窃的终结者

Chrome 146（Windows 版）正式实装 **Device Bound Session Credentials (DBSC)**，将用户身份验证会话绑定到物理设备并加密。这项技术从架构层面杜绝了 Cookie 盗窃攻击，后端只需增加注册/刷新接口即可集成，前端无需改动。macOS 版预计未来几周上线。

### 2. Temporal API 全面落地 — Date 对象时代进入倒计时

Temporal API 在 Chrome 134+ 和 Firefox 136+ 已完整支持，W3Schools 已收录完整文档。作为 TC39 Stage 3→4 的重量级提案，它提供了时区感知、日历系统、持续时间计算等原生能力，全面替代已有 28 年历史的 `Date` 对象。

### 3. React 19 Hooks 大升级 — 表单交互模式重定义

`useActionState`、`useOptimistic`、`useFormStatus` 三大新 Hook 正式落地，配合 React Compiler（Forget）的自动记忆化，大量 `useMemo`/`useCallback`/`memo` 代码可直接删除。表单状态管理从"手动拼装"进化为"声明即用"。

## 🔥 分类详解

### JavaScript / ECMAScript

#### `using` 关键字 (ES2026 Explicit Resource Management)
JavaScript 首个资源管理关键字，离开作用域自动调用 `[Symbol.dispose]()`，告别 try...finally。
```javascript
using resource = acquireResource();
resource.doSomething();
// 离开作用域自动释放，无需 finally
```

#### `Promise.try()` (ES2026)
统一包装同步/异步函数，同步异常也转为 rejected Promise。
```javascript
return Promise.try(() => maybeSyncOrAsync());
```

#### Temporal API
全面替代 Date，支持时区、日历、持续时间。
```javascript
const now = Temporal.Now.zonedDateTimeISO("Asia/Shanghai");
const duration = start.until(end);
console.log(duration.total('hours'));
```

#### Array.fromAsync()
异步版 Array.from()，直接从 Promise 数组/ReadableStream 创建数组。
```javascript
const users = await Array.fromAsync(fetchPromises);
```

### CSS 新特性

#### CSS Anchor Positioning
纯 CSS 实现 tooltip/popover 定位，浏览器自动碰撞检测与回退。
```css
.trigger { anchor-name: --my-anchor; }
.tooltip { position-anchor: --my-anchor; inset-area: top; }
```

#### CSS Custom Highlight API
编程式自定义文本高亮，无需修改 DOM。
```javascript
CSS.highlights.set("search", new Highlight(range1, range2));
```
```css
::highlight(search) { background-color: #ffeb3b; }
```

#### CSS @scope 规则
选择器限定在特定 DOM 子树，减少全局样式污染。
```css
@scope (.card) { h2 { color: #333; } }
```

#### CSS Scroll-Driven Animations
纯 CSS 滚动触发动画，合成器线程执行，主线程繁忙也 60fps。
```css
.progress { animation-timeline: scroll(root); }
```

#### LCH Colors
感知均匀颜色系统，比 HSL 更符合人眼感知，程序化生成可访问配色。
```css
--primary: lch(60% 80 270);
--primary-lg: lch(from var(--primary) calc(l + 15%) c h);
```
