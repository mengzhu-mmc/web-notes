### Web Install API（Chrome / Edge）
> 实验性 API，允许开发者通过 `navigator.install()` 以编程方式触发 PWA 安装提示，改进应用发现与分发

**浏览器支持**：Chrome 143+ / Edge 143+（Origin Trial 阶段）/ Safari 暂不支持 / Firefox 暂不支持

**代码示例**
```javascript
// 触发 PWA 安装
async function installApp() {
  try {
    const manifestId = await navigator.install();
    console.log('安装成功，manifest_id:', manifestId);
  } catch (err) {
    if (err.name === 'AbortError') {
      console.log('用户拒绝安装');
    } else if (err.name === 'DataError') {
      console.log('未找到 manifest');
    }
  }
}

// 绑定到用户交互（如按钮点击）
document.getElementById('install-btn').addEventListener('click', installApp);
```

### CSS @scope 规则（W3C 标准）
> CSS @scope 规则允许选择特定 DOM 子树中的元素，减少选择器与 DOM 结构的耦合，无需编写过于具体的选择器

**浏览器支持**：Chrome 118+ / Firefox 128+ / Safari 17.4+

**代码示例**
```css
/* 使用 @scope 限定样式作用域 */
@scope (.card) {
  :scope {
    background-color: #f5f5f5;
    padding: 16px;
    border-radius: 8px;
  }

  /* 只影响 .card 内部的 h2，不影响外部 */
  h2 {
    color: #333;
    margin-bottom: 8px;
  }

  /* 使用 .lower-bound 限制作用范围下界 */
  .content p {
    line-height: 1.6;
  }
}

/* 不同作用域互不干扰 */
@scope (.sidebar) {
  h2 {
    color: #666;
    font-size: 14px;
  }
}
```

### CSS Custom Highlight API（W3C 标准）
> 通过 JavaScript 创建文本范围，使用 CSS 定义样式来高亮文档中任意文本，适用于搜索高亮、代码标注等场景

**浏览器支持**：Chrome 105+ / Edge 105+ / Safari 17.2+ / Firefox 暂不支持

**代码示例**
```javascript
// 使用 CSS Custom Highlight API 高亮文本
const body = document.body;
const range = document.createRange();
const textNode = body.querySelector('p').firstChild;

// 定位要高亮的文本
const startIndex = textNode.textContent.indexOf('重要内容');
const endIndex = startIndex + '重要内容'.length;
range.setStart(textNode, startIndex);
range.setEnd(textNode, endIndex);

// 创建高亮对象并注册
const highlight = new Highlight(range);
CSS.highlights.set('search-result', highlight);
```

```css
/* CSS 定义高亮样式 */
::highlight(search-result) {
  background-color: #ffeb3b;
  color: #000;
}
```

### TC39 Declarations in Conditionals 提案（Stage 1）
> 允许在条件语句（if/while）中声明变量，类似 C++ 的惯用模式，简化变量声明与条件检查的组合写法

**浏览器支持**：暂未实现，当前处于 Stage 1（提案阶段）

**代码示例**
```javascript
// 当前写法（需要两步）
const result = maybeGetValue();
if (result !== null) {
  // 使用 result
  console.log(result);
}

// 提案写法：在条件中声明变量
if (const result = maybeGetValue()) {
  // result 在此作用域内可用
  console.log(result);
}

// while 循环中也适用
while (const chunk = await stream.read()) {
  process(chunk);
}
```
