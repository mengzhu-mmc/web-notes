### [CSS 锚点定位 Anchor Positioning]（Chrome 已支持）
> 纯 CSS 实现 tooltip/下拉菜单的自动定位和碰撞检测，无需 JavaScript，终结 Popper.js 依赖。

**浏览器支持**：Chrome 125+ ✅ / Firefox 开发中 🔄 / Safari 开发中 🔄

**代码示例**
```css
/* 定义锚点 */
.trigger-button {
  anchor-name: --my-anchor;
}

/* 相对于锚点定位 */
.tooltip {
  position: absolute;
  position-anchor: --my-anchor;
  inset-area: top;
  margin-bottom: 8px;
}

/* 自动回退：空间不足时翻转到另一侧 */

  position-try-fallbacks:
    flip-block,
    flip-inline;
}
```

### [CSS Subgrid]（全浏览器支持）
> 子 grid 继承父级轨道，实现卡片列表、产品对比表等场景的跨组件完美对齐。

**浏览器支持**：Chrome 117+ ✅ / Firefox 71+ ✅ / Safari 16+ ✅

**代码示例**
```css
/* 父级网格 */
.card-row {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 24px;
}

/* 卡片继承父级行轨道 */
.card {
  display: grid;
  grid-template-rows: subgrid;
  grid-row: span 3;
}

.card-header { grid-row: 1; }
.card-body   { grid-row: 2; }
.card-footer { grid-row: 3; }
```

### [Trusted Types]（2026 年全浏览器覆盖）
> CSP Level 3 原生安全特性，强制管控 innerHTML/eval 等高危 DOM 操作，从底层阻断 DOM XSS 攻击。Firefox 148+ 刚刚加入支持。

**浏览器支持**：Chrome 83+ ✅ / Firefox 148+ ✅ / Safari 17.4+ ✅

**代码示例**
```javascript
// 创建安全策略
const policy = trustedTypes.createPolicy('securePolicy', {
  createHTML: (input) => {
    return input
      .replace(/&/g, '&amp;')
      .replace(/</g, '&lt;')
      .replace(/>/g, '&gt;');
  },
  createScript: () => { throw new Error('禁止执行动态脚本'); },
});

// 安全使用 innerHTML
element.innerHTML = policy.createHTML(userInput);
```
