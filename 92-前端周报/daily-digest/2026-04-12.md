### Device Bound Session Credentials (DBSC)（Chrome 146）

> 将用户身份验证会话绑定到物理设备并加密，从根本上防御 Cookie 盗窃攻击，有望让 Cookie 劫持走向历史

**浏览器支持**：Chrome 146+（Windows 版已实装）/ macOS 版预计未来几周上线 / Firefox 暂不支持 / Safari 暂不支持

**代码示例**
```javascript
// 网站后端注册 DBSC（服务器端逻辑）
// 1. 用户登录成功后，服务器返回 RegistrationResponse
// 2. Chrome 自动将会话凭据绑定到当前设备硬件密钥

// 前端无需额外代码，浏览器自动处理：
// - 凭据加密存储在设备端
// - Cookie 自动轮换（无需手动 refresh token 逻辑）
// - 网站仍可像以前一样使用标准 Cookie

// 服务器端注册接口示例（Node.js）
app.post('/register-session', async (req, res) => {
  const { session_id } = req.body;
  // 返回 DBSC 注册挑战
  res.json({
    session_id,
    // Chrome 将自动处理凭据绑定
  });
});

// 服务器端刷新接口（替代传统 refresh token）
app.post('/refresh-session', async (req, res) => {
  // Chrome 自动携带绑定的凭据
  // 验证设备绑定后签发新 Cookie
  res.json({ status: 'ok' });
});
```

### Temporal API — JavaScript 日期时间革命（TC39 Stage 3→4）

> 全面替代 Date 对象的新日期时间 API，支持时区、日历、持续时间等原生操作，W3Schools 已收录完整文档，Chrome/Edge/Firefox 已完整支持

**浏览器支持**：Chrome 134+ / Edge 134+ / Firefox 136+ / Safari 暂不支持（开发中）

**代码示例**
```javascript
// 创建即时时间
const now = Temporal.Now.instant();
console.log(now.toString()); // "2026-04-12T03:00:00Z"

// 从 ISO 字符串创建
const meeting = Temporal.PlainDateTime.from("2026-04-15T14:30:00");
console.log(meeting.year, meeting.month, meeting.day); // 2026 4 15

// 时区感知的时间转换
const taipeiTime = Temporal.Now.zonedDateTimeISO("Asia/Taipei");
const nyTime = taipeiTime.withTimeZone("America/New_York");
console.log(taipeiTime.toString()); // "2026-04-12T11:00:00+08:00[Asia/Taipei]"
console.log(nyTime.toString());     // "2026-04-11T23:00:00-04:00[America/New_York]"

// 计算持续时间
const start = Temporal.Instant.from("2026-04-12T00:00:00Z");
const end = Temporal.Instant.from("2026-04-13T00:00:00Z");
const duration = start.until(end);
console.log(duration.total('hours')); // 24

// PlainDate 操作（无时区）
const birthday = Temporal.PlainDate.from("1990-06-15");
const today = Temporal.Now.plainDateISO();
const age = birthday.until(today);
console.log(`年龄：${age.years} 岁`);

// 日期加减
const nextWeek = today.add({ days: 7 });
const lastMonth = today.subtract({ months: 1 });
```

### CSS Custom Highlight API（W3C 标准）

> 编程式自定义文本高亮，无需修改 DOM 结构即可对任意文本范围应用样式，适用于搜索高亮、协作编辑、代码审查等场景

**浏览器支持**：Chrome 105+ / Edge 105+ / Safari 17+ / Firefox 暂不支持

**代码示例**
```javascript
// 第一步：创建 Range 对象
const parentNode = document.getElementById("article");
const range1 = new Range();
range1.setStart(parentNode, 10);
range1.setEnd(parentNode, 20);

const range2 = new Range();
range2.setStart(parentNode, 40);
range2.setEnd(parentNode, 60);

// 第二步：创建 Highlight 对象（多个 range 可共享同一高亮）
const searchHighlight = new Highlight(range1, range2);

// 第三步：注册到 HighlightRegistry
CSS.highlights.set("search-result", searchHighlight);

// 可以为不同用户注册不同高亮（协作编辑场景）
const user1Highlight = new Highlight(range1);
const user2Highlight = new Highlight(range2);
CSS.highlights.set("user-1", user1Highlight);
CSS.highlights.set("user-2", user2Highlight);
```

```css
  background-color: rgba(255, 0, 0, 0.3);

```

```javascript
// 动态搜索高亮完整示例
const input = document.getElementById("search-input");
input.addEventListener("input", () => {
  const query = input.value.toLowerCase();
  const text = document.getElementById("article").textContent.toLowerCase();

  // 清除旧高亮
  CSS.highlights.clear();

  if (!query) return;

  // 查找所有匹配位置
  const ranges = [];
  let pos = 0;
  while ((pos = text.indexOf(query, pos)) !== -1) {
    const range = new Range();
    const textNode = document.getElementById("article").firstChild;
    range.setStart(textNode, pos);
    range.setEnd(textNode, pos + query.length);
    ranges.push(range);
    pos += query.length;
  }

  // 注册新高亮
  CSS.highlights.set("search-result", new Highlight(...ranges));
});
```

### React 19 Hooks 大升级（React 官方）

> useActionState / useOptimistic / useFormStatus 三大新 Hook 重新定义表单交互模式，配合 React Compiler（Forget）自动记忆化，useMemo/useCallback 大量场景可删除

**浏览器支持**：与 React 运行时一致（所有现代浏览器）

**代码示例**
```javascript
// useActionState — 表单状态一体化管理
function ProfileForm() {
  const [error, submitAction, isPending] = React.useActionState(
    async (prevState, formData) => {
      const result = await saveUserProfile(formData);
      if (!result.ok) return "保存失败，请重试";
      return null; // null 表示成功
    },
    null // 初始状态
  );

  return (
    <form action={submitAction}>
      <input name="username" required />
      <button type="submit" disabled={isPending}>
        {isPending ? "保存中..." : "保存"}
      </button>
      {error && <p className="error">{error}</p>}
    </form>
  );
}
```

```javascript
// useOptimistic — 乐观更新，瞬间响应用户操作
function LikeButton({ initialLikes }) {

    initialLikes,
    (currentLikes, increment) => currentLikes + increment
  );

}
```
