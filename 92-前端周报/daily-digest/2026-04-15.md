### 1. CSS 锚点定位（Anchor Positioning）（Chrome/Firefox）

> 纯 CSS 实现 tooltip、dropdown 等浮层 UI 的智能定位，无需 JavaScript 库，自动处理碰撞检测和回退定位。

**浏览器支持**：Chrome 125+ / Firefox 132+ / Safari 技术预览版

**代码示例**
```css
/* 定义锚点元素 */
.trigger-button {
  anchor-name: --my-anchor;
}

/* 相对于锚点定位 */
.tooltip {
  position: absolute;
  position-anchor: --my-anchor;
  inset-area: top;
  margin-bottom: 8px;
}

/* 自动回退定位策略 */
.tooltip {
  position-try-fallbacks: 
    flip-block,   /* 上方放不下则翻到下方 */
    flip-inline;  /* 左右两侧尝试 */
}
```

### 2. CSS 瀑布流布局（Masonry Layout）（Chrome/Firefox）

> 原生 CSS 实现瀑布流布局，无需 JavaScript 计算，支持响应式和动态内容。

**浏览器支持**：Chrome 125+（需开启标志） / Firefox 132+

**代码示例**
```css
.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  grid-template-rows: masonry; /* 核心属性 */
  gap: 16px;
  align-tracks: start;
}

/* 横向瀑布流 */
.gallery-horizontal {
  grid-template-columns: masonry;
  grid-template-rows: repeat(4, auto);
  masonry-direction: column;
}
```

### 3. 滚动驱动动画（Scroll-Driven Animations）（Chrome/Firefox）

> 基于 CSS 的滚动动画，运行在 compositor thread，即使 JS 繁忙也能保持 60fps。

**浏览器支持**：Chrome 115+ / Firefox 125+ / Safari 技术预览版

**代码示例**
```css
/* 阅读进度条 */
.progress-bar {
  position: fixed;
  top: 0;
  height: 4px;
  background: linear-gradient(90deg, #0066ff, #00ccff);
  transform-origin: 0%;
  animation: grow-progress linear both;
  animation-timeline: scroll(root);
}

@keyframes grow-progress {
  from { transform: scaleX(0); }
  to { transform: scaleX(1); }
}

/* 视差效果 */
.background-layer {
  animation: parallax linear both;
  animation-timeline: scroll();
}

@keyframes parallax {
  to { transform: translateY(calc(50% * var(--parallax-speed))); }
}
```

**参考链接**
- [Scroll-driven Animations - MDN](https://developer.mozilla.org/en-US/docs/Web/CSS/scroll-timeline)
- [Chrome Developers 介绍](https://developer.chrome.com/articles/scroll-driven-animations/)

### 5. CSS Subgrid（正式可用）

> 子网格继承父级轨道，解决设计系统中组件对齐难题。

**浏览器支持**：Chrome 117+ / Firefox 71+ / Safari 16+

**代码示例**
```css
/* 产品对比表 - 完美对齐 */
.comparison-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: auto auto 1fr auto;
  gap: 2px;
}

.product-column {
  display: grid;
  grid-template-rows: subgrid;
  grid-row: 1 / -1;
  border: 1px solid #e0e0e0;
}

.product-name { grid-row: 1; }
.price { grid-row: 2; }
.features { grid-row: 3; }
.cta-button { grid-row: 4; }
```

### 6. ECMAScript Realm Snapshot（Stage 1 提案）

> 允许序列化 JavaScript 运行时状态，用于快速冷启动、Worker 通信等场景。

**提案阶段**：Stage 1（早期阶段）

**代码示例（概念性）**
```javascript
// 序列化当前 Realm 状态
const snapshot = Realm.createSnapshot();

// 在 Worker 中恢复
const worker = new Worker('worker.js');
worker.postMessage({ snapshot });

// 快速冷启动
Realm.restoreSnapshot(snapshot);
```
