## 📚 面试相关知识点摘录

> 详细笔记见 `notes/interview/` 目录

### 1. Canvas 数据压缩（新颖考题）

- 原理：利用 `canvas.toBlob()` + 图片编码，将任意二进制/文本数据转换为图片格式传输
- 场景：前端上传、存储、传输中的轻量压缩
- 相关 API：`canvas.getContext('2d')`、`ImageData`、`toBlob()`、`createImageBitmap()`

### 2. Node.js 文件系统监听（Chokidar）

**面试常问**：`fs.watch` 的缺陷是什么？

| 问题 | fs.watch 原生 | Chokidar |
|------|--------------|---------|
| 跨平台一致性 | 差（Mac/Linux/Windows 行为不同） | 好 |
| 事件防抖 | 无 | 内置 |
| 子目录递归 | 不稳定 | 稳定 |
| rename 事件 | 不可靠 | 可靠 |

**使用场景**：Vite 热更新、webpack watch、文件同步工具

### 3. 大模型与前端的结合（AI 面试方向）

- 大模型本质：统计规律匹配，而非真正推理
- 无头应用（Headless）趋势：前端 UI 与数据层分离，适配 AI 中介层
- Prompt Engineering 在前端领域的实践价值

### 4. 跨平台 UI 框架原理

**Valdi（Snapchat）核心思路**：
- 类 React 语法（组件 + 状态） → 编译器翻译 → iOS/Android/macOS 原生 Widget
- 对比：React Native（JS Bridge）、Flutter（Dart → Skia Canvas）、Valdi（类 React → 原生组件）

### 5. npm 镜像与私有包管理

- 私有 NPM 的必要场景：企业内部 SDK、monorepo 包管理
- 常见方案：Verdaccio（Node.js）、Nexus、**Npflared（Cloudflare Worker）**
- `.npmrc` 配置 scope 映射到私有源
