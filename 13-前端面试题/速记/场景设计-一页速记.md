# 场景设计一页速记

> 来源：[牛客网-面试题-场景设计.md](../牛客网-面试题-场景设计.md) | 20 题浓缩

---

## 数据结构与算法类

| 场景 | 核心技术 | 关键点 |
|---|---|---|
| **虚拟列表** | 可视区域 + 滚动偏移 | 计算 startIndex/endIndex + padding 撑开滚动条 |
| **大文件上传** | 分片 + MD5 + 断点续传 | `File.slice()` + 并发控制 + 服务端合并 |
| **LRU 缓存** | Map 有序特性 | `get` 时移到末尾，`put` 超限时删头部 |
| **并发请求控制** | Promise 并发池 | 用 `Promise.race` 控制最大并发数 |
| **数组扁平化** | 递归/迭代/reduce | 注意循环引用和最大深度 |
| **JSON.stringify** | 递归 + 类型处理 | 处理循环引用、undefined、Symbol、Date、RegExp |

## 设计模式与架构类

| 场景 | 核心技术 | 关键点 |
|---|---|---|
| **事件总线** | Map 存储监听器 | `on/once/off/emit`，`once` 用包装函数自动移除 |
| **compose/pipe** | `reduceRight/reduce` | 函数组合，从右到左/从左到右依次执行 |
| **埋点 SDK** | `sendBeacon` + 批量队列 | 页面卸载时用 `sendBeacon` 保证发送成功 |
| **错误监控** | 错误采集 + 采样 + 面包屑 | `window.onerror` + `unhandledrejection` + 性能数据 |
| **权限控制** | 路由守卫 + 动态路由 | 按钮级权限：自定义指令 / HOC 包装 |

## React 原理类

| 场景 | 核心技术 | 关键点 |
|---|---|---|
| **useState** | Hook 数组 + 索引 | 每个组件维护 hooks 链表，按调用顺序索引 |
| **useEffect** | deps 比较 + 清理函数 | 上次 deps 与本次比较，不同则执行清理 + 新 effect |
| **call/apply/bind** | Symbol 挂载 + `new.target` | `call`：在目标对象上临时挂载方法并调用 |

## 组件设计类

| 场景 | 核心技术 | 关键点 |
|---|---|---|
| **Toast 组件** | `createRoot` 命令式调用 | 队列管理 + 自动消失 + 动画 |
| **图片懒加载** | `IntersectionObserver` | 进入视口时替换 `data-src` → `src`，支持预加载 |
| **URL 参数解析** | `URLSearchParams` | 处理同名参数（返回数组） |
| **模板字符串解析** | 正则 + `new Function` | 替换 `${expr}` 为对应值 |

## 框架原理类

| 场景 | 核心技术 | 关键点 |
|---|---|---|
| **简易 Promise** | 状态机 + `queueMicrotask` | pending→fulfilled/rejected，then 链式调用 |
| **Vue 响应式** | Proxy + track/trigger | `track` 收集依赖，`trigger` 触发更新 |
