# CSS 居中方案大全

> 面试高频题：「说说 CSS 水平垂直居中的方法」，至少要能说出 4~5 种并知道适用场景。

---

## 水平居中

### 行内元素 / 文字

```css
.parent {
  text-align: center;
}
```

### 块级元素（已知宽度）

```css
.child {
  width: 200px;
  margin: 0 auto;
}
```

### Flex 容器内

```css
.parent {
  display: flex;
  justify-content: center;
}
```

---

## 垂直居中

### 单行文字（已知高度）

```css
.parent {
  height: 60px;
  line-height: 60px; /* line-height = height */
}
```

### Flex 容器内

```css
.parent {
  display: flex;
  align-items: center;
}
```

---

## 水平 + 垂直居中（5 种方案对比）

### ① Flex（推荐，最简洁）

```css
.parent {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

✅ 优点：代码简洁，子元素不需要任何配置  
✅ 兼容性：现代浏览器全部支持  
⚠️ 注意：子元素宽高不固定也能用

---

### ② Grid

```css
.parent {
  display: grid;
  place-items: center; /* align-items + justify-items 简写 */
}
```

✅ 比 Flex 更简洁  
✅ 适合整页居中布局

---

### ③ absolute + transform（不知道子元素宽高）

```css
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

✅ 子元素宽高未知也能用  
✅ 兼容性好  
⚠️ 父元素需要 `position: relative`  
⚠️ `translate` 会触发合成层，轻微性能开销

---

### ④ absolute + margin: auto（需要知道宽高）

```css
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 0; right: 0; bottom: 0; left: 0;
  width: 200px;
  height: 100px;
  margin: auto;
}
```

✅ 经典方案，兼容性极佳  
❌ 必须设置宽高

---

### ⑤ absolute + 负 margin（需要知道宽高）

```css
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  width: 200px;
  height: 100px;
  margin-top: -50px;  /* height / 2 */
  margin-left: -100px; /* width / 2 */
}
```

❌ 必须知道宽高，计算繁琐，现代项目已不推荐

---

## 方案对比

| 方案 | 子元素宽高未知 | 代码量 | 兼容性 | 推荐度 |
|------|:---:|:---:|:---:|:---:|
| Flex | ✅ | ★★★ | IE11+ | ⭐⭐⭐⭐⭐ |
| Grid | ✅ | ★★★★ | IE不支持 | ⭐⭐⭐⭐ |
| absolute + transform | ✅ | ★★ | IE9+ | ⭐⭐⭐⭐ |
| absolute + margin: auto | ❌ | ★★ | IE8+ | ⭐⭐⭐ |
| absolute + 负margin | ❌ | ★ | IE8+ | ⭐⭐ |

---

## 面试回答思路

> 「请说说 CSS 水平垂直居中的方法」

1. **Flex 是现代首选**：两行搞定，子元素不需要宽高
2. **Grid `place-items: center`**：更简洁，适合页面级布局
3. **absolute + transform**：不知道宽高时的经典方案
4. **absolute + margin: auto**：老项目兼容性要求高时用
5. 扩展聊聊：行内元素用 `text-align + line-height`，table-cell 方案（已过时）

**面试加分点**：主动提到各方案的适用场景和局限性，不只是背方案。
