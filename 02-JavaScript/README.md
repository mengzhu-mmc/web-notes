## 📖 学习路径

### 入门（1-2 周）
1. `01-数据类型与变量/` — 类型判断、隐式转换
2. `02-函数与作用域/` — 函数基础、闭包
3. `03-对象与原型/` — 对象操作、原型链

### 进阶（2-3 周）
4. `04-数组/` — 数组方法大全
5. `05-异步编程/` — Promise、EventLoop
6. `06-ES6+新特性/` — 解构、展开、迭代器、ES2020+

### 高级（2 周）
7. `07-DOM与浏览器/` — DOM 操作、事件
8. `08-模块化/` — 模块化规范
9. `10-性能与优化/` — V8 引擎、性能优化

### 实战
10. `11-实战Demo/` — 图片懒加载、树结构转换等
11. `课程笔记/` — 系统课程学习
