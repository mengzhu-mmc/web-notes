# ES2020+ 新特性

> ES2020 到 ES2024 重要新特性速览

---

## ES2020（ES11）

### 可选链 `?.`
```js
const name = user?.profile?.name;
// 等价于：user && user.profile && user.profile.name

arr?.[0];           // 数组安全访问
fn?.();             // 函数安全调用
```

### 空值合并 `??`
```js
const val = input ?? 'default';
// 仅当 input 为 null/undefined 时用默认值
// 与 || 的区别：0、''、false 不会被 ?? 替换
```

### Promise.allSettled
```js
// 等待所有 Promise 完成，不管成功还是失败
const results = await Promise.allSettled([p1, p2, p3]);
// [{ status: 'fulfilled', value: ... }, { status: 'rejected', reason: ... }]
```

### BigInt
```js
const big = 9007199254740991n;
typeof big;  // 'bigint'
```

### globalThis
```js
// 统一的全局对象引用（浏览器=window，Node=global）
globalThis.setTimeout;
```

### matchAll
```js
const str = 'test1 test2';
[...str.matchAll(/t(e)(st)/g)];
// [['test1', 'e', 'st1'], ['test2', 'e', 'st2']]
```

## ES2021（ES12）

### 逻辑赋值运算符
```js
a ||= b;  // a || (a = b)
a &&= b;  // a && (a = b)
a ??= b;  // a ?? (a = b)
```

### Promise.any
```js
// 返回第一个成功的 Promise，全部失败则抛 AggregateError
const result = await Promise.any([p1, p2, p3]);
```

### 数字分隔符
```js
const billion = 1_000_000_000;
const hex = 0xFF_FF_FF;
```

### replaceAll
```js
'hello world'.replaceAll('o', '0');  // 'hell0 w0rld'
```

## ES2022（ES13）

### 类字段与私有属性
```js
class Counter {
  count = 0;          // 公开实例字段
  #secret = 'hidden'; // 私有字段（# 前缀）

  #increment() { this.count++; }  // 私有方法
}
```

### 静态块
```js
class MyClass {
  static cache = new Map();
  static {
    // 类加载时执行一次
    this.cache.set('key', initValue());
  }
}
```

### at() 方法
```js
arr.at(-1);   // 最后一个元素
str.at(-1);   // 最后一个字符
```

### Object.hasOwn
```js
Object.hasOwn(obj, 'key');  // 替代 hasOwnProperty
```

## ES2023（ES14）

### 数组 findLast / findLastIndex
```js
[1, 2, 3, 4].findLast(n => n > 2);       // 4
[1, 2, 3, 4].findLastIndex(n => n > 2);  // 3
```

### 数组 toSorted / toReversed / toSpliced / with
```js
// 不修改原数组，返回新数组
[3, 1, 2].toSorted();     // [1, 2, 3]
[1, 2, 3].toReversed();   // [3, 2, 1]
[1, 2, 3].with(1, 99);    // [1, 99, 3]
```

### Hashbang 语法
```js
#!/usr/bin/env node
// Node 脚本首行支持
```

## ES2024（ES15）

### Object.groupBy / Map.groupBy
```js
const people = [
  { name: 'Alice', age: 25 },
  { name: 'Bob', age: 30 },
  { name: 'Carol', age: 25 }
];

Object.groupBy(people, p => p.age);
// { 25: [Alice, Carol], 30: [Bob] }
```

### Promise.withResolvers
```js
const { promise, resolve, reject } = Promise.withResolvers();
// 外部持有 resolve/reject，适合回调转 Promise
```

### ArrayBuffer 转移
```js
const transferred = ab.transfer();
// 零拷贝转移 ArrayBuffer 内容
```

### 正则 v 标志
```js
// Unicode 属性转义的增强版本
/^\p{Script=Han}+$/v.test('中文');
```
