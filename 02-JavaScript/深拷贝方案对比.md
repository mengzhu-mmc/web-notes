# 深拷贝方案对比：structuredClone vs lodash.cloneDeep

## structuredClone（原生 API）

`structuredClone` 是浏览器原生 API（Chrome 98+, Node 17+），支持更多原生类型：`Date`、`RegExp`、`Map`、`Set`、`ArrayBuffer`、`Blob`、`Error` 等，原生支持循环引用处理，性能通常更好（尤其大数据量）。

不支持的类型：函数（抛出错误）、DOM 节点（抛出错误）、Symbol（抛出错误）。

## lodash.cloneDeep

兼容性好，支持旧环境。对函数会直接引用（浅拷贝），Symbol 会被忽略。但对 `Map`/`Set` 的支持有限（可能被转为空对象）。

## 对比总结

| 特性 | structuredClone | lodash.cloneDeep |
|------|----------------|-----------------|
| Map/Set | 完美支持 | 可能转为空对象 |
| 函数 | 抛出错误 | 浅拷贝引用 |
| 循环引用 | 原生支持 | 支持 |
| 性能 | 原生 API，通常更快 | JS 实现 |
| 兼容性 | Chrome 98+, Node 17+ | 支持旧环境 |

## 深拷贝手写实现（含循环引用）

```js
function deepClone(target, cache = new Map()) {
  // 基本类型直接返回
  if (typeof target !== 'object' || target === null) return target;
  // 循环引用检查
  if (cache.has(target)) return cache.get(target);
  // 特殊类型
  if (target instanceof Date) return new Date(target);
  if (target instanceof RegExp) return new RegExp(target.source, target.flags);
  // 初始化（先存再递归，处理循环引用）
  const cloneResult = Array.isArray(target) ? [] : {};
  cache.set(target, cloneResult);
  for (let key in target) {
    if (target.hasOwnProperty(key)) {
      cloneResult[key] = deepClone(target[key], cache);
    }
  }
  return cloneResult;
}
```

## 使用建议

```javascript
// 现代项目推荐 structuredClone
const clone1 = structuredClone(data);

// 需要兼容旧环境或处理函数时用 lodash
import cloneDeep from 'lodash/cloneDeep';
const clone2 = cloneDeep(data);

// 兼容方案
const myClone = (obj) => {
  if (typeof structuredClone !== 'undefined') {
    try { return structuredClone(obj); }
    catch { return cloneDeep(obj); }
  }
  return cloneDeep(obj);
};
```

## 对象扁平化（flatten）

将嵌套对象拍平为点分路径形式。

```js
function flatten(obj, prefix = '') {
  const result = {};
  for (const key in obj) {
    if (!obj.hasOwnProperty(key)) continue;
    const newKey = prefix ? `${prefix}.${key}` : key;
    if (typeof obj[key] === 'object' && obj[key] !== null && !Array.isArray(obj[key])) {
      Object.assign(result, flatten(obj[key], newKey));
    } else {
      result[newKey] = obj[key];
    }
  }
  return result;
}

// 示例
flatten({ a: { b: { c: 1 }, d: 2 }, e: 3 });
// → { 'a.b.c': 1, 'a.d': 2, 'e': 3 }
```

## 数组转树形结构

### 递归版（简洁但 O(n²)）

```js
function arrayToTree(arr, parentId = null) {
  return arr
    .filter(item => item.parentId === parentId)
    .map(item => ({
      ...item,
      children: arrayToTree(arr, item.id)
    }));
}
```

### Map 优化版（O(n)，面试推荐）

```js
function arrayToTree(arr) {
  const map = new Map();
  const roots = [];
  // 第一遍：建索引
  arr.forEach(item => map.set(item.id, { ...item, children: [] }));
  // 第二遍：挂载
  arr.forEach(item => {
    const node = map.get(item.id);
    if (item.parentId === null) roots.push(node);
    else map.get(item.parentId)?.children.push(node);
  });
  return roots;
}

// 示例数据
const arr = [
  { id: 1, parentId: null, name: '根' },
  { id: 2, parentId: 1, name: '子1' },
  { id: 3, parentId: 1, name: '子2' },
  { id: 4, parentId: 2, name: '孙1' },
];
// arrayToTree(arr) → [{ id:1, children: [{id:2, children:[{id:4,...}]}, {id:3,...}] }]
```

**核心优化点**：用 Map 存 id → node 的映射，把两层嵌套循环降为两次线性遍历，时间复杂度从 O(n²) 降为 O(n)。
