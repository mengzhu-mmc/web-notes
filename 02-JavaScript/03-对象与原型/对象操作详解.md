# 对象操作详解

> JavaScript 对象属性描述符、常用方法与高级操作

---

## 属性描述符

```js
const obj = { name: 'Tom' };

Object.getOwnPropertyDescriptor(obj, 'name');
// { value: 'Tom', writable: true, enumerable: true, configurable: true }
```

| 描述符 | 含义 |
|---|---|
| `value` | 属性值 |
| `writable` | 是否可修改 |
| `enumerable` | 是否可枚举（for...in / Object.keys） |
| `configurable` | 是否可删除/修改描述符 |

### 定义属性
```js
Object.defineProperty(obj, 'age', {
  value: 18,
  writable: false,       // 只读
  enumerable: false,     // 不可枚举
  configurable: false    // 不可删除/修改
});
```

## 对象冻结与密封

| 方法 | 不能添加 | 不能删除 | 不能修改值 | 不能修改描述符 |
|---|---|---|---|---|
| `preventExtensions` | ✅ | ❌ | ❌ | ❌ |
| `seal` | ✅ | ✅ | ❌ | ❌ |
| `freeze` | ✅ | ✅ | ✅ | ✅ |

```js
Object.isFrozen(obj);
Object.isSealed(obj);
Object.isExtensible(obj);
```

## 常用方法

### 创建
```js
Object.create(proto);         // 指定原型创建
Object.assign(target, ...src); // 浅合并
{ ...obj1, ...obj2 };         // 展开浅合并
```

### 查询
```js
Object.keys(obj);       // 可枚举自有属性名
Object.values(obj);     // 可枚举自有属性值
Object.entries(obj);    // [key, value] 数组
Object.getOwnPropertyNames(obj);  // 所有自有属性名（含不可枚举）
Object.getOwnPropertySymbols(obj); // Symbol 属性
Reflect.ownKeys(obj);   // 所有自有键（字符串 + Symbol）
```

### 判断
```js
Object.is(a, b);    // 严格相等（处理 NaN 和 -0）
obj.hasOwnProperty('key');  // 自有属性（已废弃，推荐 Object.hasOwn）
Object.hasOwn(obj, 'key');  // ES2022
'key' in obj;       // 自有 + 原型链
```

## Getter / Setter

```js
const obj = {
  _name: 'Tom',
  get name() { return this._name; },
  set name(val) { this._name = val.toUpperCase(); }
};

obj.name = 'jerry';
console.log(obj.name);  // 'JERRY'
```

## 对象扁平化（面试常考）

```js
function flatten(obj, prefix = '') {
  const result = {};
  for (const key in obj) {
    const newKey = prefix ? `${prefix}.${key}` : key;
    if (typeof obj[key] === 'object' && obj[key] !== null && !Array.isArray(obj[key])) {
      Object.assign(result, flatten(obj[key], newKey));
    } else {
      result[newKey] = obj[key];
    }
  }
  return result;
}

flatten({ a: { b: { c: 1 } }, d: 2 });
// { 'a.b.c': 1, d: 2 }
```
