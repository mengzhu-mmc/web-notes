# 命令模式 Command Pattern

> 将请求封装为对象，支持撤销、重做、队列等操作

---

## 核心思想

```js
class Command {
  execute() { throw new Error('未实现'); }
  undo() { throw new Error('未实现'); }
}

class LightOnCommand extends Command {
  constructor(light) { super(); this.light = light; }
  execute() { this.light.on(); }
  undo() { this.light.off(); }
}

class LightOffCommand extends Command {
  constructor(light) { super(); this.light = light; }
  execute() { this.light.off(); }
  undo() { this.light.on(); }
}

// 调用者
class RemoteControl {
  constructor() { this.history = []; }

  execute(command) {
    command.execute();
    this.history.push(command);
  }

  undo() {
    const cmd = this.history.pop();
    cmd?.undo();
  }
}
```

## 前端常见场景

### 1. 撤销/重做（富文本编辑器）
```js
class Editor {
  constructor() {
    this.content = '';
    this.history = [];
    this.redoStack = [];
  }

  execute(command) {
    command.execute(this);
    this.history.push(command);
    this.redoStack = []; // 新操作清空重做栈
  }

  undo() {
    const cmd = this.history.pop();
    cmd?.undo(this);
    this.redoStack.push(cmd);
  }

  redo() {
    const cmd = this.redoStack.pop();
    cmd?.execute(this);
    this.history.push(cmd);
  }
}

// 具体命令
class InsertTextCommand {
  constructor(text, position) {
    this.text = text;
    this.position = position;
  }
  execute(editor) {
    editor.content = editor.content.slice(0, this.position) + this.text + editor.content.slice(this.position);
  }
  undo(editor) {
    editor.content = editor.content.slice(0, this.position) + editor.content.slice(this.position + this.text.length);
  }
}
```

### 2. 快捷键系统
```js
const commands = {
  'ctrl+s': () => saveDocument(),
  'ctrl+z': () => editor.undo(),
  'ctrl+y': () => editor.redo(),
  'ctrl+c': () => copySelection(),
  'ctrl+v': () => pasteClipboard()
};

document.addEventListener('keydown', (e) => {
  const key = `${e.ctrlKey ? 'ctrl+' : ''}${e.key}`;
  const cmd = commands[key];
  if (cmd) {
    e.preventDefault();
    cmd();
  }
});
```

### 3. 任务队列
```js
class TaskQueue {
  constructor() { this.queue = []; this.running = false; }

  add(command) {
    this.queue.push(command);
    if (!this.running) this.runNext();
  }

  async runNext() {
    this.running = true;
    while (this.queue.length > 0) {
      const cmd = this.queue.shift();
      await cmd.execute();
    }
    this.running = false;
  }
}

// 使用：批量上传
const queue = new TaskQueue();
files.forEach(file => {
  queue.add({ execute: () => uploadFile(file) });
});
```

## 优点
- **解耦**：调用者和接收者完全解耦
- **可组合**：支持宏命令（多个命令组合）
- **可撤销**：天然支持 undo/redo
- **可持久化**：命令对象可序列化存储
