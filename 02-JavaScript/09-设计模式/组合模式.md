# 组合模式 Composite Pattern

> 将对象组合成树形结构，统一处理单个对象和组合对象

---

## 核心思想

```js
class Component {
  constructor(name) { this.name = name; }
  add(child) { throw new Error('未实现'); }
  remove(child) { throw new Error('未实现'); }
  getChild(i) { throw new Error('未实现'); }
  operation() { throw new Error('未实现'); }
}

// 叶子节点
class File extends Component {
  operation() { return `文件: ${this.name}`; }
}

// 组合节点
class Folder extends Component {
  constructor(name) { super(name); this.children = []; }

  add(child) { this.children.push(child); }
  remove(child) { this.children = this.children.filter(c => c !== child); }
  getChild(i) { return this.children[i]; }

  operation() {
    const results = this.children.map(c => c.operation());
    return `文件夹 ${this.name}: [${results.join(', ')}]`;
  }
}

// 使用
const root = new Folder('根目录');
root.add(new File('readme.md'));
const src = new Folder('src');
src.add(new File('index.js'));
src.add(new File('app.js'));
root.add(src);
console.log(root.operation());
```

## 前端常见场景

### 1. 虚拟 DOM 树
```js
// React/Vue 的虚拟 DOM 就是组合模式
const vdom = {
  tag: 'div',
  props: { className: 'container' },
  children: [
    { tag: 'h1', props: {}, children: [{ tag: 'text', value: '标题' }] },
    { tag: 'p', props: {}, children: [{ tag: 'text', value: '内容' }] }
  ]
};

// 统一渲染：不区分元素和文本
function render(node) {
  if (node.tag === 'text') return document.createTextNode(node.value);
  const el = document.createElement(node.tag);
  node.children?.forEach(child => el.appendChild(render(child)));
  return el;
}
```

### 2. 树形菜单/组织架构
```js
class MenuItem {
  constructor(label) { this.label = label; }
  render() { return `<li>${this.label}</li>`; }
}

class Menu extends MenuItem {
  constructor(label) { super(label); this.children = []; }

  add(item) { this.children.push(item); }

  render() {
    const childHtml = this.children.map(c => c.render()).join('');
    return `<li>${this.label}<ul>${childHtml}</ul></li>`;
  }
}

// 使用
const menu = new Menu('文件');
menu.add(new MenuItem('新建'));
menu.add(new MenuItem('打开'));
const edit = new Menu('编辑');
edit.add(new MenuItem('复制'));
edit.add(new MenuItem('粘贴'));
menu.add(edit);
```

### 3. 表单组件树
```js
class FormField {
  constructor(name) { this.name = name; }
  validate() { return true; }
}

class FormGroup extends FormField {
  constructor(name) { super(name); this.fields = []; }

  add(field) { this.fields.push(field); }

  validate() {
    return this.fields.every(f => f.validate());
  }
}

// 嵌套表单校验
const form = new FormGroup('用户信息');
form.add(new FormField('用户名'));
const address = new FormGroup('地址');
address.add(new FormField('省'));
address.add(new FormField('市'));
form.add(address);
form.validate(); // 递归校验所有字段
```

## 优点
- 统一接口：客户端无需区分叶子和组合
- 易于扩展：新增组件类型不影响现有代码
- 天然适合树形结构

## 适用场景
- 虚拟 DOM
- 树形菜单/导航
- 表单组件
- 文件系统模拟
