# 前端设计模式速查

> 前端开发中最常用的 10 种设计模式

---

## 模式速览

| 模式 | 一句话 | 典型场景 |
|---|---|---|
| **单例** | 全局唯一实例 | Vuex/Redux Store、全局配置 |
| **工厂** | 统一创建对象 | 组件工厂、请求实例 |
| **观察者** | 一对多通知 | Vue 响应式、事件总线 |
| **策略** | 算法可替换 | 表单校验、排序、动画 |
| **装饰器** | 动态增强功能 | HOC、防抖、缓存 |
| **代理** | 控制对象访问 | Vue3 响应式、权限控制 |
| **适配器** | 接口转换 | 数据格式适配、兼容层 |
| **模板方法** | 固定流程+可变步骤 | 表单提交、构建流程 |
| **职责链** | 依次处理请求 | Koa 中间件、错误处理 |
| **组合** | 树形统一处理 | 虚拟 DOM、树形菜单 |

## 模式关系

```
创建型：单例、工厂
行为型：观察者、策略、命令、模板方法、职责链
结构型：装饰器、代理、适配器、组合
```

## 选择指南

- **需要唯一实例？** → 单例
- **需要灵活创建对象？** → 工厂
- **需要状态变化自动通知？** → 观察者
- **同一行为多种实现？** → 策略
- **不修改原代码添加功能？** → 装饰器
- **需要控制访问/延迟加载？** → 代理
- **接口不兼容？** → 适配器
- **流程固定但步骤可变？** → 模板方法
- **多个处理者依次处理？** → 职责链
- **树形结构统一操作？** → 组合
