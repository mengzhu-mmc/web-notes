# 手写防抖与节流

> 高频考题，几乎必考。防抖 = 最后一次触发执行；节流 = 固定频率执行。

---

## 防抖（debounce）

**场景**：搜索框输入、窗口 resize 后处理、按钮防重复点击

**核心**：在 wait 时间内再次触发则重置计时器，等待结束才执行。

```js
function debounce(fn, wait) {
  let timer = null;
  return function (...args) {
    clearTimeout(timer);
    timer = setTimeout(() => {
      fn.apply(this, args);
    }, wait);
  };
}
```

### 立即执行版（首次立即触发，之后防抖）

```js
function debounce(fn, wait, immediate = false) {
  let timer = null;
  return function (...args) {
    if (immediate && !timer) {
      fn.apply(this, args);
    }
    clearTimeout(timer);
    timer = setTimeout(() => {
      timer = null;
      if (!immediate) fn.apply(this, args);
    }, wait);
  };
}
```

## 节流（throttle）

**场景**：滚动监听、鼠标移动、游戏中的射击频率限制

**核心**：固定时间间隔内只执行一次，多余的调用丢弃。

### 定时器版（尾部执行）

```js
function throttle(fn, wait) {
  let timer = null;
  return function (...args) {
    if (!timer) {
      timer = setTimeout(() => {
        fn.apply(this, args);
        timer = null;
      }, wait);
    }
  };
}
```

### 时间戳版（首部执行）

```js
function throttle(fn, wait) {
  let lastTime = 0;
  return function (...args) {
    const now = Date.now();
    if (now - lastTime >= wait) {
      lastTime = now;
      fn.apply(this, args);
    }
  };
}
```

### 首尾都执行版（结合两种）

```js
function throttle(fn, wait) {
  let timer = null;
  let lastTime = 0;
  return function (...args) {
    const now = Date.now();
    const remaining = wait - (now - lastTime);
    if (remaining <= 0) {
      if (timer) { clearTimeout(timer); timer = null; }
      lastTime = now;
      fn.apply(this, args);
    } else if (!timer) {
      timer = setTimeout(() => {
        lastTime = Date.now();
        timer = null;
        fn.apply(this, args);
      }, remaining);
    }
  };
}
```

## 面试常见追问

| 问题 | 答案要点 |
|------|---------|
| 防抖和节流的区别？ | 防抖：最后一次触发后 wait 才执行；节流：每隔 wait 最多执行一次 |
| 防抖如何支持立即执行？ | 加 `immediate` 参数，首次调用立即执行，之后 wait 内不再触发 |
| lodash 的 `_.debounce` 有哪些参数？ | `leading`（前沿执行）、`trailing`（后沿执行）、`maxWait`（最大等待时间） |
| `this` 指向怎么处理？ | 用 `fn.apply(this, args)` 保留调用者上下文 |
| React 中如何使用？ | 配合 `useRef` 存 timer，或使用 `useMemo` 包裹，避免每次渲染重建 |
