│   │   └── 配置：别名 / 代理 / 环境变量 / CSS 预处理

├── 包管理
│   ├── npm / yarn / pnpm 对比

    ├── 部署策略 / 容器化
    ├── 性能 SDK / 监控平台
    └── 依赖安装优化 / 构建策略
