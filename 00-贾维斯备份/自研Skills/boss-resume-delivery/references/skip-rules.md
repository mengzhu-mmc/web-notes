# 跳过条件参考

命中以下任意一条，跳过该岗位不投递。

## 1. 薪资不达标

按以下规则判断，命中则跳过：

| 条件 | 下限要求 | 示例 |
|------|----------|------|
| 薪资**上限 ≥ 25K** | 下限 ≥ **13K** | 13-25K ✅；12-25K ❌ |
| 薪资**上限 < 25K** | 下限 ≥ **15K** | 15-22K ✅；14-22K ❌ |
| **大厂子公司**（腾讯/字节/华为/阿里等，不限上限） | 下限 ≥ **13K** | 13-26K ✅ |

> 大厂子公司判断：公司名含腾讯、字节、抖音、华为、阿里、百度、美团、京东、网易、小米等，或明确标注为其全资/控股子公司。

## 2. 技术栈不匹配
- JD 中**只写 Vue，不提 React** → 跳过
- JD 中明确写"**硬要求：必须会 Vue**"或"**精通 Vue**"（Vue 为主导）→ 跳过
- JD 中明确要求**会使用 Flutter** → 跳过
- JD 中明确要求**熟练掌握 WebGL / Three.js / Canvas 等前端 3D 库** → 跳过

## 3. 外包/中介
- 公司行业为**人力资源服务** → 跳过
- 认证资质含**人力资源服务许可证**或**劳务派遣经营许可证** → 跳过
- 公司介绍中明确提到 **IT Staffing 外包、驻场、派遣** → 跳过
- 公司名称**模糊不清**（如"某大型互联网公司"、"某知名上市公司"、"北京某大型企业"等）→ 视为外包，跳过
- 招聘者身份为**猎头顾问**且代招公司名称模糊 → 跳过

## 4. 学历要求
- JD 明确要求 **985 / 211 院校**（硬性要求）→ 跳过
- 用户学历：普通一本，可投大专及以上要求的岗位

## 5. 面试方式限制
- 岗位注明**不接受居家办公 / 要求现场面试**，且**城市不是北京** → 跳过
- 北京的现场面试岗位可正常投递

## 判断技巧

- 优先看页面标题中的公司名，若含"某"字通常为模糊公司名
- 看"认证资质"区域是否有人力资源相关证书
- 看招聘者所属公司与岗位公司是否一致，不一致且为猎头则谨慎
- 看公司介绍中是否有"为客户提供技术人员"、"驻场"、"外包"等字样
